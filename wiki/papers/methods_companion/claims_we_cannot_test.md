# Methods Companion — Claims We Cannot Test

> **Scaffold stub.** Populated during Phase 2 dossier build.

This page lists claims the Census data explicitly does not support,
so the manuscript can acknowledge them cleanly rather than stumble
into them. Each entry pairs a claim with an honest statement of
what the data cannot say about it and a suggestion for how to handle
the gap in the manuscript.

## Format

```
### Claim: <claim>

**Why we cannot test:** <data/design constraint>

**What we can say instead:** <the narrower claim the data supports>

**How the manuscript should handle this:** <acknowledge, defer,
or reframe>
```
