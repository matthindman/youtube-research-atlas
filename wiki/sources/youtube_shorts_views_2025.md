# YouTube Help (2025) - Shorts View Counting Change

## Full Citation

YouTube Help. 2025. "Get started creating YouTube Shorts." YouTube
Help. Retrieved 2026-04-17 from
https://support.google.com/youtube/answer/10059070?hl=en

_Citation convention: this Help page is archived as `.htm` / `.mhtml`,
so citations use section locators rather than fixed page numbers._

## One-Sentence Contribution

This Help page documents a measurement-regime change for Shorts: as of
March 31, 2025, public Shorts views count starts and replays with no
minimum watch-time threshold, while "Engaged views" remains the
continuation-based metric used for YPP eligibility and Shorts ad
revenue sharing (§ Changes.to.Shorts.views.count). [🤖]

## Research Question

This is not a research study. It is platform documentation explaining
how creators can make Shorts and, crucially for measurement, how
YouTube defines Shorts view counts after the March 31, 2025 metric
change (§ Get.started.creating.YouTube.Shorts;
§ Changes.to.Shorts.views.count). [🤖]

## Data, Sample, Geography, and Period

- **Data:** Help-page documentation about Shorts creation, view-count
  definitions, analytics labels, and monetization implications
  (§ Get.started.creating.YouTube.Shorts;
  § Changes.to.Shorts.views.count). [🤖]
- **Sample:** The page does not analyze an empirical sample; it states
  platform rules for all Shorts creators using YouTube's published Help
  documentation (§ Changes.to.Shorts.views.count). [🤖]
- **Geography:** Platform-wide policy / product documentation rather
  than a country-specific rule set (§ Get.started.creating.YouTube.Shorts).
  [🤖]
- **Period:** The relevant rule change takes effect on March 31, 2025,
  and the archived page also contains a later note about image-post
  views beginning in April 2026 (§ Changes.to.Shorts.views.count). [🤖]

## Methods

The page does not present research methods. It is best read as
platform-of-record documentation for how YouTube publicly defines the
Shorts view metric, the renamed "Engaged views" metric in analytics,
and which metric still governs monetization and eligibility
requirements (§ Changes.to.Shorts.views.count). [🤖]

## Main Findings

1. Starting March 31, 2025, Shorts views count every time a Short
   starts to play or replay, with no minimum watch-time threshold
   (§ Changes.to.Shorts.views.count). [🤖]
2. YouTube says this update gives creators a more complete account of
   cross-platform and repeat-play exposure, which means post-change
   public view counts are measuring a broader event than earlier Shorts
   view counts (§ Changes.to.Shorts.views.count). [🤖]
3. The prior continuation-based metric remains available in analytics
   under the name "Engaged views," and YouTube instructs creators to use
   it when comparing Shorts performance over time
   (§ Changes.to.Shorts.views.count). [🤖]
4. The Help page says neither YPP eligibility nor Shorts ad revenue
   sharing changes with the new raw-views definition, because both
   continue to rely on "Engaged views" rather than the broader public
   Shorts view count (§ Changes.to.Shorts.views.count). [🤖]

## Limitations / Scope Conditions

This source is `platform_documentation`, not independent scholarship.
It is authoritative for what YouTube says the metric means, but not for
how the rule change affected creator strategy, audience behavior, or
the platform's aggregate attention distribution
(§ Changes.to.Shorts.views.count). [🤖]

The archived `.htm` and `.mhtml` files appear to be duplicate captures
of the same Help page rather than distinct announcements, so this card
treats them as supplementary archives for one underlying source. [🤖]

Because the page was retrieved in April 2026, it already includes a
later note about image posts in the Shorts feed. Analysts using the
March 2025 rule change should distinguish that later 2026 addition from
the original 2025 counting change (§ Changes.to.Shorts.views.count).
[🤖]

## Temporal Scope

This source marks a clear break in cross-time Shorts measurement. Work
that compares pre-March-31-2025 Shorts views to later Shorts views, or
compares post-change Shorts views to long-form views, needs explicit
metric qualification because the underlying counted event changed
materially (§ Changes.to.Shorts.views.count). [🤖]
